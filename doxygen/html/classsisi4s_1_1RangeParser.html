void sisi4s::RangeParser::parse ( )
inline

Main parser method, it will parse a comma separated string of range delimiters.

E.g. it will turn "2 ,3 , 4, 8-10" into {2,3,4,8,9,10}. Note that it is supposed to be whitespace safe.
